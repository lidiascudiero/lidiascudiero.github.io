Predictive Maintenance for Urban Mobility

A feasibility study into transforming 7.5 million telemetry logs from urban mobility assets into a proactive "predict-and-prevent" maintenance system.

GitHub Repo

Project Overview

This project tackled the challenge of transitioning urban mobility infrastructure from reactive to predictive strategies by processing a massive dataset of 7.5 million telemetry logs. Applying a behavioral lens to IoT data, the study focused on identifying the exact transition points between normal operational states and system failures in a high-noise industrial environment.

A core component of the work was a rigorous Root-Cause Analysis (RCA) designed to decode the "behavior" of the machines. By isolating pervasive maintenance noise from genuine hardware anomalies, I was able to map the true diagnostic profile of the assets.

Key Milestones

The analysis led to a pivotal discovery: failures within the infrastructure manifested as sudden and stochastic events rather than gradual degradations. This insight shifted the project's focus from time-series forecasting to Knowledge Discovery, providing the client with a precise map of unpredictable vulnerabilities.

I developed an interactive Streamlit dashboard that serves as a diagnostic tool for stakeholders. This platform transforms raw telemetry into historical insights, allowing for real-time exploration of error patterns and root-cause identification.

Beyond the predictive modeling, I provided the client with a comprehensive data governance roadmap. This included identifying data quality gaps and outlining standards for future IoT sensor integration (vibration and temperature) to enable more granular predictive capabilities.

I implemented a high-performance pipeline using DuckDB and vectorized temporal aggregations to handle 7.5M records, ensuring the scalability of the feasibility study for large-scale urban deployment.

For a deep dive into the analytical process and the architectural workflow of this feasibility study, the GitHub repository includes the description of the experimental pipeline.
